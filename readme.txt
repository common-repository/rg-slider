== Description ==

A jQuery slideshow on your website/blog made easy

Just create a slideshow, upload images, crop them on-site and include the slideshow on your page or post.

= Main Features: =

* Create multiple slideshows
* Crop images to size without the use of Photoshop or something
* Set all kinds of options (image size, image quality, slideshow animation)
* UI that's very close to the native WordPress feel.
* Available in English and Dutch (Other translations wanted!)

== Installation ==

Installing RG Slider is easy:


1. Extract rg-slider.zip and upload the folder rg-slider to the plugins directory of your Wordpress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode in your posts or pages, or use the rg-slider function in your templates.
